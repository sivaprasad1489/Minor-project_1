GroupDNA – WhatsApp Group Chat Analyzer

Transform your WhatsApp group chats into meaningful insights using Python.

📌 Overview

GroupDNA is a Python-based data analysis project that analyzes exported WhatsApp group chats and generates valuable insights about conversations, user activity, media sharing, emojis, links, and more.

This project demonstrates how Python can be used for real-world data analysis by converting raw chat data into interactive statistics and visualizations.

🚀 Features
📊 Total Messages Analysis
👥 Most Active Participants
📅 Daily & Monthly Activity Timeline
⏰ Hourly Chat Activity
😀 Emoji Usage Analysis
🔗 URL Sharing Statistics
📷 Media Message Count
☁️ Word Cloud Generation
🔥 Most Frequently Used Words
📈 Interactive Data Visualizations

📥 Dataset

Export your WhatsApp chat without media.

→ Open Group Chat
→ More Options
→ Export Chat
→ Without Media

Place the exported .txt file in the project folder.
